== Description ==

Add DUBX network to Metamask wallet shortcode and payment processor for accept payments in DUBX coins.

== Installation ==

1. Upload `pay-with-dubx.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin.
3. Place shortcode 'add_dubx_network_button' anywhere on the site.
